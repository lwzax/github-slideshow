    agricultural pest control system

    农业病虫害是严重威胁农业生产安全、人民身体健康和环境安全的有害生物，主要包括害虫、细菌性病害、真菌性病害、病毒病、线虫病以及各种杂草等等。它不但造成农业减产甚至绝收，造成产品品质下降 失去商品性。更会造成农民收入降低，甚至威胁到生命安全。所以，科学高效的防治病虫害已经成为农业生产过程中最为重要的环节。本系统将会为你及时提供全面、有效的农业病虫害防治相关资讯，系统还提供 高精度病虫害识别功能，帮助您精准的定位病虫害类型，同时根据识别结果给出科学有效的防治方法。本系统将为您的农作物保驾护航！
